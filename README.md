# Умнее всех — генератор вопросов

Сайт-компаньон для настольной игры «Умнее всех» (Cosmodrome Games). Выдаёт вопросы
в формате оригинальных карточек по 6 категориям и 3 уровням сложности, чтобы играть
дальше, когда карточки из коробки уже пройдены.

## Возможности

- 6 категорий: культура, логика, естествознание, общие знания, география, астрономия.
- 3 уровня сложности (I — лёгкий, II — средний, III — сложный), 15 карточек на категорию
  — всего **270 вопросов** с правдивыми фактами (аудитория 8–18 лет).
- Каждый вопрос: 3 варианта ответа (правильный подсвечен зелёным), объяснение и награда
  — жетоны подсказок и ходы вперёд, как в игре.
- Показанные вопросы не повторяются в рамках сессии. История хранится в `localStorage`
  до ручного сброса кнопкой «Новая игра».
- Минималистичный дизайн (белый/чёрный/красный/синий), иконки Tabler, упор на мобильные.

## Технологии

Vite + React + TypeScript, `@tabler/icons-react`. Бэкенд не нужен — все данные на фронте.

## Запуск

```bash
npm install
npm run dev      # локальный сервер разработки
npm run build    # production-сборка в dist/
npm run preview  # предпросмотр собранной версии
```

## Структура

- `src/data/*.ts` — база вопросов по категориям (по 15 карточек × 3 уровня).
- `src/data/helpers.ts` — разворачивает карточки в плоский пул вопросов и назначает награды.
- `src/hooks/useUsedQuestions.ts` — история показанных вопросов в `localStorage`.
- `src/components/` — экраны: сетка категорий, выбор уровня, карточка вопроса, сброс.
